个人技能

 HTML5 HTML 5有两大特点：首先，强化了 Web 网页的表现性能。其次，追加了本地数据库等 Web 应用的功能
 CSS3 CSS3是CSS技术的升级版本，CSS3语言开发是朝着模块化发展的。更多新的模块也被加入进来. 盒子模型、列表模块等。
 javascript JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。
 jQuery jQuery是一个快速、简洁的JavaScript框架.jQuery设计的宗旨是“write Less，Do More”
 AJAX AJAX 是一种用于创建快速动态网页的技术.通过在后台与服务器进行少量数据交换，AJAX 可以使网页实现异步更新。
 AngularJS AngularJs是由谷歌开发 ，数据绑定， DOM操作,MVC设计模式和模块加载 是通过指令扩展了HTML,前通过表达式绑定数据到HTML.
 node.js Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效
 PHP PHP是一种运行在服务器的脚本语言！是一种创建动态交互性站点的强有力的服务器端脚本语言
